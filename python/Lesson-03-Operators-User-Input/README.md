Lesson 03: Operators & User Input

Welcome to Lesson 03 of the Python Programming Class.

In this lesson, students will learn how to use operators to perform calculations
and how to collect input from users** to make Python programs interactive.

This lesson builds on:
- Lesson 01: Introduction to Python
- Lesson 02: Variables & Data Types


Lesson Objectives
By the end of this lesson, students should be able to:
- Explain what operators are
- Use arithmetic, assignment, comparison, and logical operators
- Collect user input using input()
- Convert user input to numbers using type casting
- Write simple interactive programs


Lesson Contents
- notes.md` → Detailed explanations of operators and user input
- examples.py` → Practical coding examples
- assignment.md` → Practice tasks for students
