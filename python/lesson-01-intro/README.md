Lesson 01: Introduction to Python

Welcome to Lesson 01 of the Python Programming Class.

This lesson introduces Python, its features, and basic syntax rules.
It is designed for students with no prior programming experience.


Lesson Objectives
By the end of this lesson, students should be able to:
- Explain what Python is
- State why Python is popular
- Install Python successfully
- Write and run a basic Python program
- Understand basic syntax and indentation
- Use variables


Lesson Contents
- notes.md: Detailed lesson notes
- intro.py: Practical Python examples
- assignment.md: Practice tasks for students
