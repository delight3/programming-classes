Lesson 05 Assignment: Control Flow Statements

Instructions
- Attempt all questions
- Write clean and readable code

Task 1: Simple If Statement
Write a program that:
1. Stores a number
2. Checks if the number is greater than 10
3. Prints an appropriate message


Task 2: If-Else Statement
Write a program that:
1. Stores a user’s age
2. Prints:
   - "Eligible to vote" if age ≥ 18
   - "Not eligible to vote" otherwise


Task 3: If-Elif-Else
Write a program that:
1. Stores a student’s score
2. Prints the grade based on:
   - 70 and above → A
   - 50–69 → B
   - Below 50 → C


Task 4: While Loop
Write a program that prints numbers from 1 to 10 using a while loop.


Task 5: For Loop
Write a program that prints all even numbers between 1 and 20 using a for loop.


Task 6: Break and Continue
Write a loop that:
1. Prints numbers from 1 to 10
2. Stops the loop when the number reaches 7
3. Skips printing the number 4


Submission
