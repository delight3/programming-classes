Lesson 04: Strings and Numeric Functions

Welcome to Lesson 04 of the Python Programming Class.

In this lesson, students will learn how to work with strings (text) and numeric functions
in Python. These concepts are essential for data processing, user interaction,
and calculations in real-world programs.


Lesson Objectives
By the end of this lesson, students should be able to:
- Understand what strings are
- Use common string operations and methods
- Work with numbers using built-in numeric functions
- Convert between strings and numbers
- Apply string and numeric functions in simple programs


Lesson Contents
- notes.md": Detailed explanations of strings and numeric functions
- examples.py: Practical coding examples
- assignment.md: Practice tasks for students

Start with notes.md to begin the lesson.
