Lesson 02: Variables & Data Types

Welcome to Lesson 02 of the Python Programming Class.

In this lesson, students will learn how to store data using variables
and understand different data types in Python.


Lesson Objectives
By the end of this lesson, students should be able to:
- Define variables correctly
- Follow variable naming rules
- Understand common Python data types
- Identify data types using type()
- Perform basic operations with variables
