HTML Attributes

What is an HTML Attribute?
An HTML attribute provides additional information about an HTML element.
Attributes are always written in the opening tag and usually come in
name/value pairs.

Syntax
<tagname attribute="value">Content</tagname>

Common HTML Attributes

1. id
Uniquely identifies an element.

<p id="intro">Welcome to HTML</p>

2. class
Used to group elements for styling or scripting.

<p class="text-primary">Hello World</p>

3. style
Adds inline CSS styling.

<p style="color: blue; font-size: 18px;">Styled text</p>

4. title
Displays extra information on hover.

<p title="This is a tooltip">Hover me</p>


5. href
Used in links to define the destination URL.

<a href="https://www.google.com">Visit Google</a>


6. src
Specifies the source of media files.

<img src="image.jpg" alt="Sample Image">


7. alt
Provides alternative text for images.

<img src="logo.png" alt="Company Logo">


8. disabled
Disables form elements.

<button disabled>Submit</button>


9. placeholder
Shows hint text inside input fields.

<input type="text" placeholder="Enter your name">
