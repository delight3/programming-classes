# HTML Lesson 01 – Document Structure

## Lesson Objective
Introduce students to the basic structure of an HTML document and the purpose of HTML tags.

## Topics Covered
- What HTML is and how it works
- Basic HTML document structure
- Common tags:
  - `<html>`
  - `<head>`
  - `<title>`
  - `<body>`
  - Headings (`<h1>` – `<h6>`)
  - Paragraphs (`<p>`)

- HTML tags
   1. Text Formatting Tags
   2. Links & Media Tags
   3. List Tags
   4. Table Tags
   5. Form Tags
   6. Semantic Tags  


## Lesson Files
- `index.html` – Demonstration file used during class
- `tags.html` - Demonstration file used during class
- `notes.md` – Instructor explanations and notes
- `assignment.md` – Student practice exercise


## Teaching Notes
This lesson is designed for beginners and focuses on understanding HTML layout
before introducing styling or interactivity.
