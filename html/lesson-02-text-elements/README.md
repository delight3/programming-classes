# Lesson 02: Text Elements in HTML

## Overview
This lesson introduces **HTML text elements**, which are used to structure, format, and present textual content on a web page. Understanding these elements is essential for creating readable, accessible, and well-structured websites.

By the end of this lesson, you will be able to properly use HTML tags to display headings, paragraphs, formatted text, quotes, code snippets, and lists.


## Learning Objectives
After completing this lesson, you should be able to:
- Use headings to organize content correctly
- Create paragraphs and line breaks
- Format text using inline elements
- Display quotations and citations
- Show code and preformatted text
- Create ordered and unordered lists
- Apply semantic text elements for accessibility

## Topics Covered
HTML tags

### 1. Headings
HTML provides six levels of headings used to define content hierarchy:
- `<h1>` to `<h6>`

**Best practice:** Use only one `<h1>` per page.


### 2. Paragraphs & Line Breaks
Used to structure blocks of text:
- `<p>` — paragraph
- `<br>` — line break
- `<hr>` — horizontal rule

### 3. Text Formatting Elements
Used to emphasize or modify text:
- `<strong>` / `<b>` — bold text
- `<em>` / `<i>` — italic text
- `<u>` — underline
- `<mark>` — highlighted text
- `<small>` — smaller text
- `<sup>` / `<sub>` — superscript and subscript
- `<del>` / `<ins>` — deleted and inserted text

### 4. Quotes & Citations
Used to represent quotations:
- `<blockquote>` — block quotation
- `<q>` — inline quotation
- `<cite>` — citation

### 5. Code & Preformatted Text
Used for displaying code examples:
- `<code>` — inline code
- `<pre>` — preformatted text
- `<kbd>` — keyboard input
- `<samp>` — sample output

### 6. Lists
Used to organize content:
- `<ul>` — unordered list
- `<ol>` — ordered list
- `<li>` — list item
- `<dl>`, `<dt>`, `<dd>` — description list

### 7. Links Inside Text
Used to navigate between pages:
- `<a href="">` — anchor tag
- `target="_blank"` — open link in new tab
- `title` attribute — tooltip text


### 8. Semantic Text Elements
Improve meaning and accessibility:
- `<abbr>` — abbreviation
- `<address>` — contact information
- `<time>` — date and time
- `<span>` — inline container


## Accessibility & Best Practices
- Use semantic tags instead of styling text
- Avoid excessive use of `<br>` for spacing
- Structure content logically for screen readers
- Use `<strong>` and `<em>` for meaning, not just appearance


## Practice Task
Create a simple web page that includes:
- A main heading and subheadings
- At least two paragraphs
- One quote and one citation
- A code example using `<pre>` and `<code>`
- An ordered and an unordered list
