# Lesson 02 Assignment: HTML Text Elements

## Instructions
- Create an HTML file named `assignment.html`.
- Use the tags you have learned in Lesson 02.
- Follow proper HTML structure: `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`.
- Use meaningful headings and semantic tags.
- Add comments in your code explaining each section.
- Save your work and submit the HTML file.

## Task 1: Headings and Paragraphs
1. Create a main heading (`<h1>`) for your page: **"My Favorite Hobby"**.
2. Add two subheadings (`<h2>`): **"Introduction"** and **"Why I Enjoy It"**.
3. Under each subheading, write 2â€“3 paragraphs (`<p>`) describing your hobby.
4. Add a horizontal line (`<hr>`) between sections.

## Task 2: Text Formatting
1. Highlight important words in your paragraphs using `<strong>` or `<em>`.
2. Use `<mark>` to highlight your favorite aspect of your hobby.
3. Include one superscript (`<sup>`) and one subscript (`<sub>`) in your text.
4. Use `<small>` to add a footnote or extra comment.

## Task 3: Quotes and Citations
1. Add a blockquote (`<blockquote>`) with a quote about hobbies.
2. Include an inline quote (`<q>`) somewhere in your paragraphs.
3. Cite the source of your quote using `<cite>`.

## Task 4: Code Example
1. Include a code snippet related to your hobby using `<pre>` and `<code>`.
   - Example: If your hobby is programming, show a small code snippet.

2. Use `<kbd>` to show a keyboard shortcut and `<samp>` to show sample output.

## Task 5: Lists
1. Create an unordered list (`<ul>`) of tools or items you use for your hobby.
2. Create an ordered list (`<ol>`) showing steps to perform your hobby.
3. Optional: Use a description list (`<dl>`) to explain terms related to your hobby.

## Task 6: Links
1. Add a link (`<a>`) to a website or resource related to your hobby.
2. Open the link in a new tab using `target="_blank"`.
3. Add a `title` attribute to the link describing the destination.

## Task 7: Semantic Text Elements
1. Use `<abbr>` to explain at least one abbreviation in your text.
2. Include `<time>` to mention when you usually practice your hobby.
3. Use `<address>` to show contact info if applicable.

## Bonus Task
- Add comments in your HTML explaining what each tag is doing.
- Make your page neat, readable, and well-structured.

**Submission:** Submit your completed `assignment.html` file.

**Evaluation Criteria:**
- Proper use of HTML tags
- Correct headings and paragraph structure
- Appropriate text formatting
- Proper use of quotes, code, and lists
- Clean and readable code with comments
- Bonus points for semantic and accessible HTML
