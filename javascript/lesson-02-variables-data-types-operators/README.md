# Lesson 02: Variables, Data Types, and Operators

## Overview
This lesson introduces **JavaScript variables, data types, and operators**, and builds practical skills using **HTML Canvas**.  
Learners will create shapes, understand basic animation concepts, animate a moving ball, and learn how to **debug JavaScript code**.


## Learning Objectives
By the end of this lesson, learners should be able to:
- Declare and use variables correctly
- Identify JavaScript data types
- Apply arithmetic and logical operators
- Draw shapes on HTML Canvas
- Understand animation concepts and timers
- Animate a moving ball on the screen
- Debug JavaScript programs effectively


## Topics Covered
- Variables in JavaScript
- Data Types
- JavaScript Operators
- Creating Shapes in Canvas
- Introduction to Animation and Timers
- Animating Ball Movement on Screen
- Debugging JavaScript


## Requirements
- Web Browser (Chrome, Firefox, Edge)
- Code Editor (VS Code recommended)
- Basic knowledge of JavaScript and HTML Canvas


 How to Run
1. Clone or download the repository
2. Open `index.html` in your browser
3. Open Developer Tools → Console for debugging


## Files Explained
- `index.html` – HTML structure and canvas element
- `script.js` – JavaScript logic and animation code
- `NOTE.md` – Detailed explanations and examples
- `README.md` – Lesson overview and usage
