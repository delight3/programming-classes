# Lesson 01 Notes – Introduction to JavaScript

## 1. Introduction to JavaScript
JavaScript is a programming language used to make web pages interactive.  
It allows websites to respond to user actions such as clicks, typing, and scrolling.

### Key Points
- Runs in the browser
- Works with HTML and CSS
- Used for frontend and backend development


## 2. JavaScript and the Document Object Model (DOM)
The **DOM (Document Object Model)** represents an HTML document as a tree of objects.

JavaScript uses the DOM to:
- Read HTML elements
- Modify content
- Change styles
- Handle user events

Every HTML element becomes an object in JavaScript.


## 3. Accessing an Object in the DOM
Common methods used to access HTML elements:


document.getElementById("id");
document.getElementsByClassName("class");
document.getElementsByTagName("tag");
document.querySelector("selector");
document.querySelectorAll("selector");
