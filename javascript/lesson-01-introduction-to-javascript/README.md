# Lesson 01: Introduction to JavaScript

## Overview
This lesson introduces the fundamentals of **JavaScript**, its interaction with the **Document Object Model (DOM)**, basic object access, an introduction to **jQuery**, and **HTML Canvas**.  
Learners will also build their first **Hello World JavaScript application**.


## Learning Objectives
By the end of this lesson, learners should be able to:
- Explain what JavaScript is
- Understand the Document Object Model (DOM)
- Access and manipulate DOM objects
- Access object attributes
- Understand the basics of jQuery
- Explain HTML Canvas and its coordinate system
- Build a simple “Hello World” JavaScript app


## Topics Covered
- Introduction to JavaScript  
- JavaScript and the Document Object Model (DOM)  
- Accessing an Object in the DOM  
- Accessing Object Attributes  
- Introduction to jQuery  
- Introduction to HTML Canvas  
- Canvas Coordinate System  
- “Hello World” App in JavaScript  


## Requirements
- Web Browser (Chrome, Firefox, Edge)
- Code Editor (VS Code recommended)
- Basic HTML knowledge


## How to Run
1. Clone or download the repository
2. Open `index.html` in a web browser
3. Open Developer Tools → Console to view outputs


## Files Explained
- `index.html` – Main HTML file
- `script.js` – JavaScript logic
- `NOTE.md` – Detailed lesson notes and explanations
- `README.md` – Project overview and instructions
