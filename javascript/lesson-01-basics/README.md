JavaScript – Lesson 01: Basics
Description

Lesson 01 – JavaScript Basics introduces the fundamental concepts of JavaScript.
This lesson is designed for beginners who want to understand how JavaScript works and how it adds interactivity to web pages.


Objectives

By the end of this lesson, you will be able to:

Understand what JavaScript is and where it runs

Declare and use variables

Work with basic data types

Use operators

Write simple functions

Output data to the browser and console

Topics Covered

What is JavaScript?

Linking JavaScript to HTML

Variables (var, let, const)

Data Types

Operators

Console logging
