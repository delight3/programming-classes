Notes (Important for Beginners)
What is JavaScript?

JavaScript is a programming language used to make web pages interactive.
It runs in the browser and responds to user actions.

Variables

var – old method (avoid using)

let – value can change

const – value cannot change

let score = 10;
const pi = 3.14;

Data Types

String - "Hello"

Number - 25

Boolean - true / false

Null - null

Undefined - undefined


Operators
+  addition
-  subtraction
*  multiplication
/  division

Console

Use the console to debug and test your code.
console.log("Testing JavaScript");
